# 接线说明 (Wiring Guide)

## 所需硬件

- ESP32-S3-DevKitC-1 开发板
- DHT11 温湿度传感器模块
- BH1750FVI 光照传感器模块（GY-302）
- MQ-135 空气质量传感器模块
- SSD1306 0.96" OLED 显示屏（I2C，128x64）
- 面包板 + 杜邦线
- 4.7KΩ 电阻（DHT11 上拉）

## 详细接线图

```
ESP32-S3 DevKitC-1
┌─────────────────────────────┐
│                             │
│  3V3 ──┬── BH1750 VCC      │
│        ├── SSD1306 VCC      │
│        └── DHT11 VCC        │
│                             │
│  GND ──┬── BH1750 GND      │
│        ├── SSD1306 GND      │
│        ├── DHT11 GND        │
│        └── MQ135 GND        │
│                             │
│  5V  ──── MQ135 VCC         │
│                             │
│  GPIO21 (SDA) ──┬── BH1750 SDA   │
│                 └── SSD1306 SDA  │
│                             │
│  GPIO22 (SCL) ──┬── BH1750 SCL   │
│                 └── SSD1306 SCL  │
│                             │
│  GPIO4 ──── DHT11 DATA     │
│           └── 4.7KΩ ── 3V3 │
│                             │
│  GPIO1 (ADC1_CH0) ── MQ135 AOUT │
│                             │
└─────────────────────────────┘
```

## 注意事项

1. **DHT11 上拉电阻**: DATA 引脚必须接 4.7KΩ 上拉电阻到 3.3V
2. **MQ135 供电**: 必须接 5V，3.3V 会导致读数不准
3. **I2C 总线**: BH1750 和 SSD1306 共享 SDA/SCL，注意地址不要冲突
4. **预热时间**: MQ135 首次使用需要预热 24-48 小时
5. **I2C 上拉**: 大多数模块自带上拉电阻，如通信不稳定可外接 4.7KΩ
